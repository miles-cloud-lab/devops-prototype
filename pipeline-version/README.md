# DevOps 流水线版本构建增强原型

## 当前原型

- `index.html`: 合并后的统一原型，页面内支持切换单分支模式和多分支模式。
- `multi-branch-tag.html`: 兼容旧入口的跳转页，会自动跳到统一原型的多分支视图。

## 设计依据

- 用户提供的 UI 规范
- 参考截图：Flux DevOps 流水线流程详情页、运行流水线弹窗

## 本页表达范围

- 保持现有 DevOps 流水线流程详情页结构。
- 在同一个原型页面中切换单分支模式和多分支模式。
- 单分支模式：在运行配置弹窗中二选一选择分支或标签，且只能保留一条运行来源。
- 多分支模式：每条运行来源都可独立选择分支或标签，并支持继续添加新引用行。
