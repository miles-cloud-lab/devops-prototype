# Flux SonarQube 代码扫描原型

## 当前原型

- `index.html`: SonarQube 代码扫描原型，覆盖代码扫描列表、新建扫描配置、启动扫描、扫描记录、扫描配置详情、流水线任务抽屉和报警通知配置。

## 设计依据

- PRD: `PRD-SonarQube-Code-Scan.md`
- 来源：Flux DevOps 接入 SonarQube 代码扫描需求讨论、现有 Flux DevOps 代码管理/流水线/报警通知页面截图

## 本页表达范围

- 代码扫描作为 `Flux DevOps > 代码管理 > 代码扫描` 独立菜单。
- 新建扫描配置时选择代码仓库，一个代码仓库对应一个 SonarQube 项目。
- 代码扫描列表展示仓库、路径、关联应用、拥有人、创建时间、触发规则和操作。
- 支持手动启动扫描、查看扫描记录，并跳转 SonarQube 查看详情。
- 扫描配置页支持关联应用、语言、扫描路径、触发规则、质量门禁和通知模板配置。
- 流水线 SonarQube 代码扫描任务保留原抽屉结构，移除通知人配置，支持质量门禁不通过时是否阻断流水线。
- 报警通知中将代码扫描通知与流水线运行通知分开配置。

## 暂不表达

- SonarQube 原生规则、质量门禁和问题详情页面。
- SonarQube 项目历史数据迁移。
- 修改已绑定代码仓库。
- 权限同步失败后的后台重试和排查页面。
- 多语言插件的完整兼容性验证。
