# DevOps AI 辅助代码评审原型

## 当前原型

- `index.html`: AI 辅助代码评审原型，覆盖流水线任务配置、代码评审列表和 diff 详情页。

## 设计依据

- PRD: `PRD-AI-Code-Review.md`
- 来源：DevOps 代码评审需求讨论与现有 Flux DevOps 评审流程

## 本页表达范围

- 保持现有 Flux DevOps 流水线代码评审任务结构。
- 在代码评审任务配置中展示 `AI 辅助评审` 开关。
- 在代码评审列表中展示 AI 评审状态、风险等级、问题数和 AI 摘要。
- 在 diff 详情页展示 AI 总览、文件风险标识、行内 AI 评论和右侧评审面板。
- 表达手动重新评审、采纳、忽略、误报和转人工确认等反馈动作。

## 暂不表达

- AI 模型调用、prompt 编排和上下文截断策略。
- 后端任务调度、结果落库和重试机制。
- AI 结果作为流水线阻断条件的配置。
- 跨项目规则学习、团队知识库接入和自动修复 Agent。
