# DevOps 流水线构建版本追溯原型

## 当前原型

- `index.html`: 流水线最近运行页代码源卡片和分支详情弹窗原型。

## 设计依据

- 保持 Panther / Flux DevOps 现有页面风格。
- 仅参考云效代码源卡片和分支详情弹窗的信息组织方式。

## 本页表达范围

- 单代码源和多分支模式的代码源卡片保持一致交互。
- 卡片保留原有代码源摘要、耗时和日志入口。
- 通过“分支详情”弹窗查看构建分支、分支明细、commitId 和提交说明。
- commitId 默认展示短 Commit，hover 展示完整 Commit。

## 暂不表达

- 流水线名称动态变化。
- 临时发布分支或集成分支的显式展示。
- 自动回滚。
- 跨仓库依赖关系可视化。
