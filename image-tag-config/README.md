# DevOps 镜像构建任务 Tag 配置原型

## 当前原型

- `index.html`: 镜像构建任务编辑抽屉原型，支持在任务级配置独立镜像 Tag。

## 设计依据

- PRD: `PRD-Image-Build-Task-Tag-Config.md`
- 来源：用户提供的 PRD 与现状原型材料

## 本页表达范围

- 保持现有 Flux DevOps 流水线流程详情页结构。
- 在编辑任务抽屉中展示 `镜像 Tag` 字段。
- 覆盖通用构建、Dockerfile 构建、两阶段构建三类镜像构建任务。
- 支持可输入下拉框、流程变量选择、手动输入固定值或变量表达式。
- 展示问号 hover 提示和不填写时继承流程配置默认镜像后缀的规则。

## 暂不表达

- 后端 API 字段落库与运行时解析逻辑。
- 构建日志与运行详情中的最终镜像展示。
- 真实流程变量接口、变量校验、镜像 Tag 合法性校验。
- 历史流水线兼容迁移实现。
